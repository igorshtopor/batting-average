###### Batting Average

Install dependancies

`bundle install`

Run tests 

`rake test`

Run server

`bundle exec ruby app.rb`

Run client

`ruby client.rb -f 'samples/Batting.csv' -y 1871 -t 'Fort Wayne Kekiongas'`

`-f` is for passing input csv file with players/batting.

`-y` is for year filtering.

`-t` is for team name.
